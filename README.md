# chatNUS Project - An Overview

Welcome to **chatNUS**, a chatbot targeted at users who prefer text-based todo lists!

## Setting up Duke

### Prerequisites

JDK 11, any IDE that supports Java, Gradle

### Running Duke

From the command line, run the following:

```
gradle run
```
